# Laboratórios de Fundamentos de Python para PDI 🐍🖼️

Este repositório contém a sequência de laboratórios práticos desenvolvidos durante a disciplina de programação no curso de **Engenharia Elétrica**. O objetivo central desta trilha foi consolidar a base lógica e o domínio da linguagem Python como pré-requisito para o estudo e aplicação de **Processamento Digital de Imagem (PDI)**.

## 🎯 Objetivo
Construir competência técnica em algoritmos e estruturas de dados, fornecendo o suporte necessário para a manipulação computacional de imagens e sinais, competências essenciais na formação de Engenharia Elétrica.

## 🛠️ Tecnologias e Ferramentas
* **Linguagem:** Python
* **Ambiente:** Jupyter Notebook / Google Colab
* **Contexto Acadêmico:** Engenharia Elétrica - Foco em Processamento de Imagens.

## 📂 Estrutura dos Laboratórios
A estrutura segue uma evolução lógica de complexidade, preparando o ambiente para operações de processamento de dados:

* **Iniciação:**
    * `lab1-introducao-python.ipynb`: Fundamentos de sintaxe e tipos de dados.
    * `lab2-Keviny Ryan.ipynb`: Implementação de lógica de controle e fluxos iniciais.

* **Estruturação de Algoritmos:**
    * `lab3 (Parte 1 e 2)` e `lab4`: Resolução de problemas estruturados e manipulação de variáveis complexas.

* **Aprofundamento Técnico:**
    * `lab5 (Parte 1 e 2)` e `lab6`: Desenvolvimento de scripts focados em eficiência lógica.
    * `lab7 (Parte 1 e 2)`: Consolidação de algoritmos para suporte a ferramentas de processamento e análise técnica.

## 🚀 Utilização
Os notebooks podem ser executados localmente via Jupyter ou através do Google Colab para reprodução dos algoritmos e testes lógicos.
